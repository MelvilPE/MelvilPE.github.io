📘 Introduction

DAVA Revanced is a reworked and enhanced continuation of DAVA.Framework .

This project provides a set of tools for modding Blitz Tank Models, allowing creators to customize and enhance game assets independently for personal use and community-driven purposes.

📘 ResourceEditor

ResourceEditor is the main tool for creating and editing Blitz Tank Models.

The original framework supported up to version 24. This version supports up to version 48.

  • Extended Features: YAML editing, global scene infos
  • Material Editor: new flags, presets, cull modes
  • Compatibility: latest tank models support

📗 Unregistered Components

Components not supported by the framework but still loaded. Editable via YAML.

📗 Marked Unregistered

Option to mark components as unregistered and switch to YAML editing. 

keyedArchive:
  list:
    variantVector:
    - string: "SlotComponent"

📗 Scene Infos

Scene Components, Render Config and Particle Emitters are now YAML-based.

Keeping these features in YAML is currently more practical due to frequent changes.

📘 FormatEditorCLI

CLI tool to convert between YAML and engine file formats.

  • KeyedArchive → YAML
  • YAML → KeyedArchive
  • Scene → YAML
  • YAML → Scene
FormatEditorCLI -i input.sc2 -o output.yaml -m 2

📘 ResourceDLC

Tool for working with DVPK / DVPD / DVPM archives.

ResourceDLC -i input.dvpm -o Output/ -m 0

📘 ResourceDVPL

Tool for working with DVPL lite archives. (not released publicly)

📘 Options Editor

This tool is available on the website and allows you to easily edit settings through sections, groups, and properties.

It is inspired by the old DevOptions menu from the game, which was removed starting from update 6.8.

Original fields were inaccessible. I rebuilt the form as a user-friendly interface faithful to the original design.

You can upload a YAML file to rebuild the menu with your properties, including any already modified ones.
Additional fields in the file should also be recognized, which is useful after developer updates.

The editor has two sections: optionsGlobal and optionsDevice.
Uploading your own YAML is recommended, as the default site version is based on optionsDesktop and the platform can be selected accordingly.

Make your changes in the interface, then export the updated YAML file.
Replace it in World Of Tanks Blitz/Data/optionsGlobal.yaml` or optionsDesktop.yaml.
Then delete the optionsGlobal.bin and optionsDesktop.bin files in the DAVAProject folder, along with any .bk files if present.

The game will regenerate the binary files with your new settings, applying the changes in-game.
